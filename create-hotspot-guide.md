# 🔥 Mac mini WiFi 熱點設定指南

> 100% 可靠的連接方案，跳過路由器限制

---

## 🎯 為什麼用熱點？

使用 Mac mini 創建 WiFi 熱點的優點：

✅ **100% 可靠** - 直接連接，無路由器干擾
✅ **無需修改設定** - 不用碰防火牆或路由器
✅ **展示專業** - 顯示完全獨立的系統架構
✅ **速度快** - 點對點連接，延遲最低

## 📋 設定步驟（5 分鐘）

### 在 Mac mini 上：

#### 步驟 1：打開系統偏好設定
1. 點擊左上角蘋果圖標
2. 系統偏好設定

#### 步驟 2：設定網際網路共享
1. 點擊「**共享**」
2. 左側列表找到「**網際網路共享**」（先不要勾選！）

#### 步驟 3：配置共享選項
1. **共享來自**: 選擇「**乙太網路**」或「**Wi-Fi**」（選擇你目前有網路的那個）
2. **使用以下連接埠共享**: 勾選「**Wi-Fi**」

#### 步驟 4：設定 WiFi 詳細資訊
1. 點擊「**Wi-Fi 選項...**」按鈕
2. 設定：
   - **網路名稱**: `FROST-Demo` （或任何你想要的名稱）
   - **頻道**: 選擇 `11`（通常干擾最少）
   - **安全性**: 選擇 `WPA2 個人級`
   - **密碼**: 設定一個簡單的密碼，例如 `bitcoin2025`
   - **確認密碼**: 再次輸入密碼
3. 點擊「**好**」

#### 步驟 5：啟用網際網路共享
1. 勾選左側的「**網際網路共享**」
2. 會彈出警告訊息：「確定要打開網際網路共享嗎？」
3. 點擊「**啟動**」

#### 步驟 6：確認啟用
狀態應該顯示：
```
網際網路共享：開啟
```

---

### 在 Surface Go 4 上：

#### 步驟 1：連接到 Mac mini 的熱點
1. 點擊右下角的 WiFi 圖標
2. 找到 `FROST-Demo`（或你設定的名稱）
3. 點擊連接
4. 輸入密碼：`bitcoin2025`
5. 等待連接成功

#### 步驟 2：確認連接
1. 打開 PowerShell 或 CMD
2. 執行：
   ```powershell
   ipconfig
   ```
3. 找到 WiFi 介面卡，應該看到類似：
   ```
   無線區域網路介面卡 Wi-Fi:
      IPv4 位址 . . . . . . . . : 192.168.2.x
      預設閘道 . . . . . . . . : 192.168.2.1
   ```

#### 步驟 3：找出 Mac mini 的新 IP
Mac mini 的熱點 IP 通常是：`192.168.2.1`

測試連接：
```powershell
ping 192.168.2.1
```

#### 步驟 4：訪問服務

在瀏覽器輸入：

**首頁：**
```
http://192.168.2.1:8000
```

**Dashboard：**
```
http://192.168.2.1:8000/dashboard.html?api=http://192.168.2.1:3000
```

**API 測試：**
```
http://192.168.2.1:3000/health
```

## 🎨 替代方案：USB 直連（終極方案）

如果 WiFi 熱點也不行，使用 USB 直連：

### 在 Mac mini 上：
1. Surface Go 4 用 USB-C 線連接到 Mac mini
2. 系統偏好設定 → 共享 → 網際網路共享
3. 共享來自：Wi-Fi
4. 使用以下連接埠共享：勾選「USB 10/100/1000 LAN」
5. 啟用網際網路共享

### 在 Surface Go 4 上：
1. Windows 會自動識別 Mac mini 為網路介面卡
2. 執行 `ipconfig` 查看分配的 IP
3. 通常 Mac mini 的 IP 是：`192.168.2.1`

## 🧪 測試連接

創建一個快速測試腳本：

**test-new-ip.bat**（在 Surface Go 4 上執行）
```batch
@echo off
echo Testing connection to Mac mini...
echo.

REM 測試熱點 IP
set MAC_IP=192.168.2.1

echo Testing ping...
ping -n 2 %MAC_IP%

echo.
echo Testing API...
curl http://%MAC_IP%:3000/health

echo.
echo Testing Dashboard...
curl -I http://%MAC_IP%:8000/

echo.
echo If all tests pass, open browser:
echo http://%MAC_IP%:8000
pause
```

## 📊 故障排除

### 問題：Mac mini 無法啟用網際網路共享

**原因：** 可能與現有網路設定衝突

**解決：**
1. 先關閉其他共享服務
2. 重新啟動 Mac mini
3. 再次嘗試

### 問題：Surface Go 4 找不到熱點

**原因：** WiFi 頻段不相容

**解決：**
1. Wi-Fi 選項中，將頻道改為 `1` 或 `6`
2. 確認 Surface Go 4 的 WiFi 已開啟
3. 重新整理 WiFi 列表

### 問題：連接後無法訪問服務

**原因：** IP 地址可能不是 192.168.2.1

**解決：**
在 Surface Go 4 上執行：
```powershell
ipconfig | findstr "閘道"
```
使用顯示的閘道 IP 替代 192.168.2.1
